# Tracking Fishing Data for the Last Year
## Function of this Project
    * This project will make analyze different instances to predict what fish you will catch
    * Instances include:
        1. Weight
        2. Length
        4. Precipitation
        5. Wind
        6. Temperature
        9. Species
    * Analyzes correlations between these instances and more:
        1. Topwater
        2. Spinning
        3. Wind
        4. Bait
## Project Usefulness
    * It will allow anglers to predict when and what they would need to catch a specific type of fish
## How to run
    1. Make sure you  run pip install pandas matplotlib seaborn scikit-learn scipy
    2. Open fishing_project.ipynb and click run all
